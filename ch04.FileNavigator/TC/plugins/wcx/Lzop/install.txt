Lzop plugin v1.0 for Total Commander

How to install Lzop plugin:

1. Unzip the LzopWincmd.wcx to the Total Commander directory
2. In Total Commander, choose Configuration - Options
3. Open the 'Packer' page
4. Click 'Configure packer extension WCXs'
5. type  lzo  as the extension
6. Click 'new type', and select the LzopWincmd.wcx
7. Click OK

What it does:

Extract files packed by Lzop:
http://www.oberhumer.com/opensource/lzop/
