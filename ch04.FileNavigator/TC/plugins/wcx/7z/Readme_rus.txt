Плагин для Total Commander.

Версия 1.0.1.0 (07/01/2005)

Предназначен для просмотра,  создания и изменения
7z-архивов.

Работает через консольную версию архиватора 7-Zip
(файл 7za.exe из дистрибутива).   оследнюю версию
этого  архиватора можно бесплатно скачать с сайта
разработчика www.7-zip.org.

Работает  довольно  медленно, но все же работает.

Также  есть  возможность  заходить  в  архивы  по
нажатию  Crtl+PgDn,  то  есть можно просматривать
самораспаковываюциеся архивы (с расширением exe).

Устанавливается  плагин  как  обычно,  в качестве
расширения  следует  указать  '7z'. Файл  7za.exe
должен  находится  в  одной  папке  с плагином, а
иначе нам удачи не видать:)

Все  замечания  и  предложения "кидайте" на ящик.
