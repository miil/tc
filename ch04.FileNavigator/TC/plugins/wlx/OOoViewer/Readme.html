Introduction

This plugin allows to view OpenOffice.org, MS Word 2007 and some other XML-based documents in Total Commander and Universal Viewer. Documents are converted to XHTML form, so you can see them with most of formatting. Not all document styles may be supported because of underlying XSL stylesheets work -- I hope, we will have full-featured XSL stylesheets someday that will support more formats and more document styles.

Currently supported formats:

  • OpenOffice.org 1.x Writer/Calc/Draw (SXW STW SXC STC SXD STD)
  • OpenOffice.org 2.x Writer (ODT OTT)
  • MS Word 2007 (DOCX DOTX)
  • FictionBook (FB2)

In Thumbnails view mode of Total Commander plugin allows to show documents thumbnails (currently only for OOo 2.x and if thumbs are embedded into document).

WARNING

Currently plugin cannot work properly in TC Lister (F3) mode (most of keys in Lister don't work while plugin is active). But it works OK in TC Quick View mode. So plugin work is disabled in Lister mode, to enable it create the "OOoViewer.ini" file in the plugin folder, and enable the option in it: 

    [Options]
    AllowWorkInLister=1
This note is not valid for the Universal Viewer (ATViewer). Plugin works OK with Universal Viewer, since keys handling in the UV is different.

Installation

With Total Commander 6.50 or newer, just open the archive and TC will install the plugin automatically. If you already have plugin installed, you may first need to uninstall the previous version so plugin's detection string will be updated.

DOCX Support

To view DOCX documents you need to install one of the following:

  • Translator from Clever Age (ODF Translator Command Line). Note that it requires the .NET Framework 2.0 to work.

  • Translator from Novell (OpenOffice OpenXML Translator). It doesn't require .NET, and it works a little faster than the one from Clever Age. Get the odfconverter*.oxt file, it is the ordinary ZIP file.

Unzip the Translator package into any subfolder of the plugin folder. The OdfConverter.exe file should appear. If you didn't unzip the Translator where needed, you'll see the error message box on opening DOCX files. If you installed the Translator right, you'll see the conversion progress window on opening DOCX.

You can also download the plugin and ODF Translator in one package.

Configuration

You can edit the following options in the "OOoViewer.ini" file, located in the plugin folder:

Key Meaning
[Options] section
AllowWorkInLister Enables plugin work in TC Lister (F3 mode).
Default is 0 (disabled, only TC QuickView is enabled).
ShowImages Enables showing images for FB2 documents. Plugin will start slower.
Default is 1 (enabled).
TempExtension Enables using extension in the temp folder's name. When enabled, folder name for "example.odt" file will look like "C:\Temp\example.odt.tmp", overwise it will look like "C:\Temp\example".
Default is 1.
Note that it must be set to 0 if you use Novell ODF Converter.

Versions History

  • 1.6.9 (08 Nov 2007):
    • Added: Each file now has its separate temporary directory, no more conflicts between several files.
    • Added: "TempExtension" ini key.
    • Fixed: FB2 images were not removed on exit.
    • Fixed: IE cache isn't used now.
    • Fixed: Memory leak on using WebBrowser (thanks Starina).
    • Fixed: View of FB2 in Win-1251 encoding (thanks Nick).

  • 1.6.5 (02 Oct 2007):
    • Added: Support for FB2.
    • Added: Support for images inside FB2. To disable images, write to the OOoViewer.ini:
      [Options]
      ShowImages=0
    • Fixed: Plugin didn't receive focus on pressing Tab in TC QuickView mode.

  • 1.5.3 (25 Jun 2007):
    • Added: Support for DOCX, DOTX.
    • Changed: Temporary files are created in the "%TEMP%\OOoUtils" folder.
    • Changed: Readme rewritten.
    • Fixed: View error with Russian plugin filepath.
    • Fixed: Conversion error of Russian filenames with Novell translator.

  • 1.1.5 (18 Jun 2007):
    • Added: Support for TC commands: Search (TC 7.0), Copy, Select All, Print.
    • Added: Extensions in the detect-string: SXC STC SXD STD
    • Changed: Disabled work in TC Lister mode, see WARNING above.
    • Fixed: Wrong detection of Lister window

  • 1.0.2 (09 Oct 2006):
    • Added: Support for OOo 1.x

  • 1.0.0 (07 Oct 2006):
    • Initial version supported OOo 2.x
